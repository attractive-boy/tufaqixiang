## 🎨 设计规范

### 颜色方案
```
主色：    #FA541C (橙色 - 用于按钮、重点)
副色：    #FF8C42 (浅橙 - 用于渐变)
背景：    #F5F5F5 (浅灰 - 页面背景)
白色：    #FFFFFF (卡片背景)
文字主：  #333333 (强调文本)
文字副：  #666666 (次要文本)
文字助：  #999999 (辅助文本)
边框：    #EEEEEE (分割线/边框)
```

### 字体大小
```
超大标题：72rpx  (品牌名称)
大标题：  48rpx  (分类标题)
标题：    32rpx  (主标题)
正文：    28rpx  (卡片内容)
小文字：  24rpx  (辅助信息)
微小：    20rpx  (时间戳、备注)
```

### 间距规范
```
大间距：40rpx  (页面上下留白)
中间距：20rpx  (卡片内部)
小间距：15rpx  (元素之间)
微间距：10rpx  (细节调整)
```

### 圆角规范
```
大圆角：20rpx   (模态框、大卡片)
中圆角：10rpx   (卡片、输入框)
小圆角：6-8rpx  (微小元素)
完全圆形：50%  (头像、圆形按钮)
```

## 🎓 使用说明

### 开发环境准备
```bash
# 1. 进入项目目录
cd /Users/licheng/Desktop/tufaqixiang/tufaqixiang

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev:h5

# 4. 打开浏览器访问
# http://localhost:8080
```

### 项目配置修改
```
修改应用名：manifest.json -> "name"
修改主色：所有页面搜索 #FA541C 替换
修改页面标题：index.html -> <title>
```

### 页面导航
```javascript
// 普通导航（保留前一页）
uni.navigateTo({ url: '/pages/forum/forum' })

// 标签页切换
uni.switchTab({ url: '/pages/forum/forum' })

// 重定向（关闭当前页）
uni.redirectTo({ url: '/pages/login/login' })
```
