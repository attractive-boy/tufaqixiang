# 兔发骑想 - Tufaqixiang

AI 规划骑行路线的综合平台，集成地图导航、社交论坛、个人中心等功能。

## 项目特性

✨ **核心功能**
- 🗺️ **路线规划**：集成高德/百度地图，支持AI智能规划
- 🤖 **AI规划**：基于用户偏好和骑行条件的智能路线推荐
- 🏙️ **北京景点**：精选北京特色骑行线路
- 💬 **社交论坛**：用户交流、经验分享、装备讨论
- 👤 **个人中心**：用户信息、骑行数据、成就勋章、收藏管理
- 📊 **数据统计**：骑行里程、完成路线、成就跟踪

## 技术实现

- **框架**：uniapp（兼容 iOS、Android、H5、小程序）
- **语言**：Vue 2/3
- **样式**：SCSS
- **响应式**：rpx 自适配所有屏幕

## 开发贴士

1. **页面导航**：使用 `uni.navigateTo()` 或 `uni.switchTab()`
2. **本地存储**：使用 `uni.setStorage()`/`uni.getStorage()`
3. **消息提示**：使用 `uni.showToast()` 或 `uni.showModal()`
4. **修改主色**：全局搜索 `#FA541C` 替换为新颜色
5. **响应式**：使用 rpx 单位而非 px
